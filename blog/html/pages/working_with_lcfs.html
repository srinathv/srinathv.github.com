Links for Jaguar/Titan

It is easier to go to one place and find useful links:

PGI Accelerator

Interactive Que

CrayPat: Cray profiler

http://www.olcf.ornl.gov/kb_articles/software-jaguar-craypat/

There is system variable that lets you change the location of .craypat, which seems to hold all the object files that have interted hooks for profiling. Search for .craypat in this webpage.

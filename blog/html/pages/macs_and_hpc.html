Macs and HPC

Macports does help

I like using macports since it really handles package dependencies well. I have had great sucess with macport’s built GNU compilers.

Making macports packages defaults

Most macports built packages have the executables namespaced as not clash with the system ones. It is nice to have the option of making a macport built package default by changing the name.
